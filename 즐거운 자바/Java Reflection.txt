1. 팩토리 메소드 패턴
 - 복잡한 생산 과정을 숨기고, 완성된 인스턴스만 반환한다. 

2. Java Reflection
구체적인 클래스 타입을 알지 못하더라도 그 클래스의 메서드, 타입, 변수들에 접근할 수 있도록 해주는 자바 API

 - 클래스 로더 - 클래스 이름을 당장 모를 때 인스턴스를 생성
 - 클래스 정보를 CLASSPATH 에서 읽어들이고 그 정보를 cl 변수가 참조하도록 한다. 

String className = "";
Class cl = cl.forName(className);
Object o = cl.newInstace();

클래스 변수로 선언된 메소드의 정보를 가져와 실행할 수 있다. - invoke()

예시) 
Spring 프레임워크의 어노테이션 같은 기능들이 리플렉션을 이용하여 
프로그램 실행 도중 동적으로 클래스의 정보를 가져와서 사용
