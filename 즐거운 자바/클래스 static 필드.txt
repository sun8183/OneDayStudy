1. 자바 JVM은 CLASSPATH( 환경변수 ) 에서 클래스를 찾아 실행한다. 

2. 클래스 정보 자체는 정적으로 메모리에 올라간다. 
=> 클래스에 static 필드가 있는지 먼저 확인한다. 
=> 클래스에 static 한 필드는 정적 영역에 따로 저장된다. 
=> 따라서 다른 클래스여도 같은 메모리 영역에 저장되어 영향을 받는다.

인스턴스를 만들지 않아도 메모리에 올라간다. 

인스턴스 필드 : static이 붙어 있지 않음  ( new 연산으로 메모리를 만들어야 사용 가능 )
클래스 필드 : static이 붙어 있음

static --> 메모리를 먼저 올린다. ( 메모리 생성 시점이 인스턴스와 다름 )

클래스 내에서 static 메소드 에서는 static 필드만 사용할 수 있다.
