생성자 : 인스턴스를 생성할 때 사용, 어떤 값을 가지고 인스턴스가 만들어지게 하고 싶다면 생성자를 사용 

특징 
1. 생성자가 없으면 인스턴스 생성시 자동으로 만들어 진다. 
2. 기본생성자는 매개변수를 하나도 받지 않는 생성자를 말한다. 

생성자 오버로딩 후 매개변수가 다른 생성자 안에서 다른 생성자를 호출할 수 있다 ( this 생성자 사용 )
EX)  this("", "", "" ) --> super 생성자를 호출하는 코드 다음이나 코드 앞에 있어야 한다. 

super() : 부모생성자 호출, 부모 상속관계에서 클래스 컴파일 시에 super() 자동으로 들어간다. 
( 부모의 기본생성자 호출 )

부모가 기본생성자가 없으면 자식 생성자는 자동으로 super() 함수를 호출, 
사용자는 반드시 직접 super() 생성자를 호출하는 코드를 작성해야 한다.
