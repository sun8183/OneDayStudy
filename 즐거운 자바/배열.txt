[ 배열 array ] 
1. 참조 타입
2. 같은 타입의 변수가 여려개 필요할 때 사용한다. 

==========================================

1. 기본형 타입의 배열은 초기값이 지정된다. 
2. 배열선언과 동시에 변수를 생성할 수 있다. 
3. null을 참조하는 변수의 length 필드를 접근하면 Exception 발생!
4. 값을 가질 수 없는 배열 (0개짜리 배열)도 존재한다. 

2차원 배열
1. 참조하는 배열의 참조하는 배열을 가리킨다. 

Arrays 
배열을 다룰 때 사용하는 유틸리티 ( java,util.Arrays )

1. copyOf : 배열을 전체 복사, copyOfRange : 지정한 범위만큼 복사 
* 같은 배열을 참조한다는 것은 아니다. 

깊은복사 VS 얕은복사 


2. compare : 배열을 비교 - int 반환 (양수, 0, 음수)

3. sort : 배열을 정렬, Object 를 정렬 할 시엔 Compareable 인터페이스의 compareTo 함수를 구현해서 사용한다.

4. binarySearch : 배열을 검색, 찾기 - 이진검색 알고리즘이 적용된 함수 

# 명령 행 아규먼트 
 - 메인메소드에 있는 args 배열 
 - JVM이 메인메소드를 실행하며 JVM이 메인메소드에 배열의 인자값을 전달한다.  

# 제한없는 아규먼트(가변) 
 - 메소드 아규먼트를 가변적으로 전달한다. 
 - int sum(int ... args)
